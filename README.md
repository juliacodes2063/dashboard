# Location-Based Forecasting Dashboard

Невеликий демо-дашборд для візуалізації прогнозу запасів та попиту з використанням лінійних графіків.

## Технології

- Next.js 16
- React 19
- Tailwind CSS 4
- ECharts 6

## Запуск проєкту

```bash
npm install
npm run dev
```

Далі відкрийте http://localhost:3000 у браузері.
